# 🧸 Cute Digital Clock App

Aesthetic and interactive digital clock built using HTML, CSS, and JavaScript.  
This project features dynamic background themes that change automatically based on the time of day, creating a cute and relaxing visual experience 🌙☀️

---

## ✨ Features

- 🕒 12-hour digital clock (AM/PM format)
- 🌈 Automatic theme changes based on time:
  - ☀️ Morning / Day and chores theme
  - ☀️ Day/ Eating theme
  - ☁️ Afternoon sky and writing theme
  - 🧸 Evening / calm and foodie theme
  - 🌙 Night / cozy night and study theme
- 🧸 Cute animated emoji character
- 🎨 Soft pastel aesthetic UI
- ⏱️ Real-time updates every second

---

## 💻 Tech Stack

- HTML
- CSS
- JavaScript
